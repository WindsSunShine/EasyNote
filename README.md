EasyNote 
==========
* 简单介绍

EasyNote to write notes

简易的便签应用，最直白的便签应用

可与Evernote笔记进行同步，已同步过的笔记右下角有云朵显示，且只会同步一次

增加对3DTouch的主屏支持(update 20160102)
